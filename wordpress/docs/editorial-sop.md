# Editorial SOP

## Roles

- Pulse Author: drafts and updates assigned content
- Pulse Editor: reviews, requests changes, schedules and publishes
- Admin/Publisher: final authority, plugin/theme and ops ownership

## Weekly Publishing Flow

1. Author creates draft (`post`, `issue`, or `event`).
2. Author sets tags/categories/section and submits for review.
3. Editor reviews:
  - headline and excerpt quality
  - SEO title/description
  - featured image alt text
  - links and dates
4. Editor schedules publish date/time.
5. Editor or publisher confirms after publication:
  - appears in relevant archive
  - menu and internal links work
  - sitemap includes new URL

## Issue Publishing SOP

1. Create new `Issue` post.
2. Fill required issue metadata:
  - season
  - status
  - issue number
  - summary
  - PDF URL
3. Attach cover image and alt text.
4. Set status:
  - `upcoming` until release
  - switch to `current` on launch
  - set previous current issue to `archived`

## Event Publishing SOP

1. Create `Event` post.
2. Set event date and location fields.
3. Add RSVP/info link.
4. Publish and verify event appears on archive view.
