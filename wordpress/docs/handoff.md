# Handoff Package

## Ownership Transfer

- Hosting platform owner
- Domain and DNS owner
- WordPress administrator accounts
- Backup/restore access
- Plugin update permissions

## Deliverables

- Theme and plugin source under `wordpress/`
- Import scripts under `wordpress/tools/`
- Runbooks and SOPs under `wordpress/docs/`
- Cutover and rollback checklist

## First 30 Days Operations

- Weekly plugin/theme update check
- Weekly backup verification
- Monthly restore drill (staging)
- Editorial workflow retro after first publication cycle
