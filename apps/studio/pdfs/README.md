# PDF Issues Folder

Copy your magazine issue PDFs here for uploading to Sanity.

## File Naming Suggested Format

`issue-{number}-{season}-{slug}.pdf`

Examples:
- `issue-01-summer-2026-signal-noise.pdf`
- `issue-02-autumn-2026-after-interface.pdf`

## How to Upload to Sanity

1. Copy PDFs to this folder
2. Open Sanity Studio: `pnpm dev:studio`
3. Navigate to: `Magazine Content → Issues`
4. Create or edit an issue
5. In the **PDF File** field, click Upload and select the PDF from this folder
6. Publish the issue

## Note

These PDFs are not committed to git (see `.gitignore`). They are only for local upload to Sanity's asset store.
